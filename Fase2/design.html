Proposito

Este proyecto de diseño web segun las hojas de estilo en cascada CSS, tiene como finalidad crear una idea central de conocimiento frente a el desarrollo y diseño de paginas web tratando de que podamos aprender todas aquellas reglas y parametros para poder fabricar productos de calidad a la vista y que sean muy maleables, utilizando las diversas herramientas como los frameworks que permiten crear un diseño universal y fácil de utilizar

Hoja de estilo CSS

En una regla de estilo, s e aplica una configuración base por defecto que permita heredar un diseño original que no será igual pero que contiene grandes fragmentos que permiten ejecutar acciones similares, pero no iguales. Colomina Pardo, O., Arques Corrales, P., & Montoyo-Bojo, J. (2011). Tema 2, parte 2: Hojas de estilo para la web. CSS. Tecnologías Web.

Frameworks

Los framework fueron creados para evitar que un desarrollador consuma más tiempo colocando reglas y demás accesos para que su creación fuera agradable y siguiera las normas pactadas, entonces se crearon unas plantillas por medio de un objeto. Lafosse, J. (2010). Struts 2: El framework de desarrollo de aplicaciones Java EE. Ediciones Eni.
